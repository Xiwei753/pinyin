# 私人数据模板

这里存放的是用户私人数据的**模板和示例**，仅用于展示数据格式。

## ⚠️ 警告
**绝对不要将你真实的个人词库、常用短语、地址、账号等信息直接写在这里并提交到 public 仓库！**

## 如何使用
1. 创建一个私有的 GitHub 仓库，用于存放你的真实数据。
2. 使用 `tools/sync/init-private-repo.sh` 初始化私人仓库。
3. 把这些模板文件复制到你的私人仓库（通常克隆在 `.private_sync/` 目录下）。
4. 在你的私人仓库里修改这些文件，填入你的真实数据。
5. 使用 `tools/sync/push-private.sh` 和 `tools/sync/pull-private.sh` 来同步你的私人数据。
